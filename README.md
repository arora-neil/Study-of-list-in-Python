# AIM: Study of List in Python

# Theory:
 append(element): Adds a single element to the end of the list.
 index(value, [start, end]): Returns the index of the first occurrence of the specified value. A ValueError is raised if the value is not found.
 insert(index, element): Inserts an element at the specified index, shifting subsequent elements to the right.
 remove(value): Removes the first occurrence of the specified value. A ValueError is raised if the value is not found.
 sort(key=None, reverse=False): Sorts the list in place in ascending order by default. You can use optional parameters like reverse=True for descending order. len(list): Returns the total number of elements in the list.
max(list): Returns the item with the maximum value in the list.
min(list): Returns the item with the minimum value in the list.
sum(list): Returns the sum of all elements in a list of numbers.
